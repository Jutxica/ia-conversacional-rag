# Infraestrutura Inicial

Este diretório contém os arquivos e configurações iniciais para o sistema de IA Conversacional com RAG.

## Componentes
1. **Backend**: APIs e integração com LLM.
2. **Frontend**: Interface de usuário.
3. **Pipeline de Dados**: Ingestão e indexação.
4. **Docker Compose**: Configuração para desenvolvimento local.

## Como Começar
1. Configure o ambiente com `docker-compose up`.
2. Siga os passos nos arquivos de configuração de cada componente.
3. Expanda a infraestrutura conforme necessário.
